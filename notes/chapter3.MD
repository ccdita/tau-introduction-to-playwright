## Chapter 3.2 - Coding Like a Pro

### Page Object Model (POM)
- POM is a representation of a page or a group of elements on a page
    - e.g., a POM can represent the header or footer of a page
- Actions and assertions can be mapped and reused within a POM
- Create POMs in ./pages/ (e.g., `./pages/home-page.ts`)
